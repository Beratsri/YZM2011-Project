# YZM2011 - Machine Learning Project

## Problem Description

Turkey's electricity producers must submit hourly generation bids to the EPİAŞ day-ahead market before 12:00 each day, committing to production for the following 24 hours. Wrong bids result in imbalance penalties or wasted capacity. This project builds a complete end-to-end ML pipeline to predict Turkey's total hourly electricity consumption (`consumption_mwh`) 24 hours in advance, and to classify whether a given hour will be a high-demand or low-demand period.

## Dataset Source

- **EPİAŞ Transparency Platform** - hourly electricity consumption and production data for Turkey (January 2021 – December 2025, ~43,824 rows)
- **Open-Meteo Archive API** - daily mean temperature, relative humidity, and wind speed for Istanbul, Ankara, and Izmir, averaged as a national proxy
- **Enriched dataset** - `main_data_enriched.csv` (generated by P1, used by P2 and P3)

## Key Findings

### P1 - Exploratory Data Analysis
- `lag_7d` (same hour last week) has r ≈ 0.88 with consumption - the strongest single predictor
- `lag_1d` (same hour yesterday) has r ≈ 0.87 - captures the daily demand pattern
- Temperature has a **U-shaped** relationship with consumption (minimum demand at ~15–18°C)
- Weekends reduce demand by ~9.4%, public holidays by ~18%, bayram by ~27%
- Strong bimodal seasonality: peaks in January–February (heating) and July–August (cooling)

### P2 - Regression Modeling
24 features engineered including cyclical sin/cos encodings for hour, month and day-of-week, polynomial temperature, and interaction terms.

| Model | CV R² | Test R² | Test RMSE |
|-------|-------|---------|-----------|
| Baseline (lag_1d only) | 0.726 | 0.742 | 3,332 MWh |
| Multiple Linear Regression | 0.899 | 0.913 | 1,933 MWh |
| Polynomial (deg=2) | −0.55 | 0.947 | 1,508 MWh |
| **Ridge (L2) ← chosen** | **0.906** | **0.912** | **1,943 MWh** |
| Lasso (L1) | 0.904 | 0.911 | 1,959 MWh |

Ridge selected for highest CV R² (0.906) - 42% RMSE improvement over baseline.

### P3 - Classification
Binary target: `high_demand = 1` if consumption > training-period median. Train: 2021–2023, Val: 2024, Test: 2025.

| Model | Val F1 | Test F1 | AUC-ROC |
|-------|--------|---------|---------|
| Logistic Regression | 0.925 | 0.928 | 0.983 |
| Decision Tree | 0.940 | 0.947 | 0.979 |
| SVM (RBF) | 0.949 | 0.950 | 0.992 |
| Random Forest | 0.959 | 0.957 | 0.994 |
| LightGBM | 0.962 | 0.963 | 0.994 |
| **XGBoost (tuned) ← chosen** | **0.963** | **0.961** | **0.994** |

XGBoost selected for highest validation F1. Only 301 / 8,760 test-set hours misclassified (3.4%).
